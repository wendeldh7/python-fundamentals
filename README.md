# Formação Python Fundamentos 🐍

Olá! 👋

Seja muito bem-vindo(a) à **Formação Python Fundamentos**. Criei este README com o objetivo de oferecer um caminho claro, direto e completo para quem deseja aprender Python do zero ou reforçar seus conhecimentos na linguagem. Python é hoje uma das linguagens mais utilizadas no mundo, presente em diversas áreas como desenvolvimento web, ciência de dados, automação e inteligência artificial.

Neste curso, você vai aprender de forma progressiva, começando pelos conceitos mais básicos até chegar a tópicos mais avançados — sempre com foco na prática e no entendimento real do que está sendo feito.

---

## 🔍 O que você vai aprender

Abaixo, explico com mais detalhes o que será abordado em cada etapa da formação:

### ✅ Módulo 1: Introdução ao Python
- O que é Python e por que aprender essa linguagem
- Como instalar o Python na sua máquina
- Configuração de um ambiente de desenvolvimento (VS Code, Jupyter, etc.)
- Escrevendo e executando seus primeiros programas

### ✅ Módulo 2: Tipos de Dados e Variáveis
- Tipos primitivos: inteiros, floats, strings, booleanos
- Conversões de tipos (casting)
- Nomeação e boas práticas com variáveis

### ✅ Módulo 3: Operadores e Expressões
- Operadores aritméticos, relacionais e lógicos
- Precedência de operadores
- Operações com strings e listas

### ✅ Módulo 4: Controle de Fluxo
- Estruturas condicionais: `if`, `elif`, `else`
- Laços de repetição: `for` e `while`
- Controle de loops com `break` e `continue`

### ✅ Módulo 5: Estruturas de Dados
- Listas e suas operações básicas
- Tuplas e como usá-las
- Dicionários e manipulação de dados
- Conjuntos (sets)

### ✅ Módulo 6: Funções
- Criando e reutilizando funções
- Parâmetros e retorno de valores
- Escopo de variáveis
- Funções anônimas (lambda)

### ✅ Módulo 7: Programação Orientada a Objetos (POO)
- Conceitos de classes e objetos
- Atributos e métodos
- Encapsulamento, herança e polimorfismo
- Criando estruturas de código mais organizadas e reutilizáveis

### ✅ Módulo 8: Manipulação de Arquivos
- Leitura e escrita de arquivos `.txt` e `.csv`
- Trabalhando com arquivos externos de forma segura
- Uso prático com leitura de dados

### ✅ Módulo 9: Conceitos Avançados
- Iteradores e geradores
- Decoradores
- Introdução ao tratamento de exceções
- Módulos e pacotes

---

## 💡 Como você vai aprender

A metodologia do curso é voltada para a **aprendizagem ativa**. Isso significa que você não apenas assiste às aulas, mas **coloca a mão na massa**:

- Vídeo-aulas objetivas com explicações claras
- Desafios e exercícios práticos em cada módulo
- Projetos reais para desenvolver seu portfólio
- Atividades que simulam situações do dia a dia profissional

---

## 🧰 Recursos disponíveis

Durante o curso, você terá acesso a:

- Material de apoio para download (slides, cheatsheets, códigos)
- Espaço para tirar dúvidas e interagir com outros alunos
- Certificados de conclusão para cada etapa vencida
- Sugestões de projetos para praticar após o curso

---

## 🎯 Ao final da formação, você será capaz de:

- Criar programas completos com Python
- Automatizar tarefas no seu trabalho ou estudos
- Organizar seu código de forma profissional com POO
- Trabalhar com dados e arquivos externos
- Utilizar recursos mais avançados da linguagem com segurança

---

Este curso foi feito para:

- Iniciantes que nunca programaram antes
- Pessoas que estão migrando para a área de tecnologia
- Profissionais que querem aprender Python para aplicar no trabalho
- Estudantes que desejam uma base sólida na linguagem
